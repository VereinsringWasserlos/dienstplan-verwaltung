# Dienstplan-Verwaltung

WordPress-Plugin zur Verwaltung von Diensten, Vereinen und Veranstaltungen für den Vereinsring Wasserlos.

## Features
- Dienst- und Slotverwaltung
- Vereins- und Bereichsmanagement
- Timeline-Übersicht
- Auto-Refresh und AJAX-Funktionen

## Installation
1. Plugin-Ordner in `wp-content/plugins/` kopieren
2. Im WordPress-Backend aktivieren

## Entwicklung
- Quellcode im Ordner `dienstplan-verwaltung`
- Hauptdatei: `dienstplan-verwaltung.php`
- Admin-Logik: `admin/`
- Öffentliche Logik: `public/`
- Datenbank: `includes/`
