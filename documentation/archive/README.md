## Archivierte Versionen

### Version 0.4.7
**Datum:** November 2025  
**Status:** Abgelöst durch 0.9.0

**Dokumente:**
- `VERSION_0.4.7_FEATURES.md` - Feature-Übersicht

**Wichtigste Änderungen in 0.4.7:**
- Admin-Modal-Funktionen implementiert
- Frontend-Dienst-Anmeldung mit Split-Funktion (alte Implementierung)
- Datenbank-Struktur-Dokumentation erstellt
- Migrations-Script für mitarbeiter_id

**⚠️ Hinweis zu 0.4.7:**
Die Split-Dienst-Funktion in Version 0.4.7 erstellt neue Dienste (Duplikate).  
Diese Implementierung wurde in Version 0.9.0 komplett überarbeitet und verwendet jetzt ein Slot-System.

## Warum archiviert?

Ab Version 0.9.0 fokussiert sich die Dokumentation auf die aktuelle Version und den Ausblick (Roadmap). Alte Versionen werden archiviert, um die Haupt-Dokumentation übersichtlich zu halten.

**Für aktuelle Dokumentation siehe:**
- `/documentation/VERSION_0.9.0_FEATURES.md` - Aktuelle Features
- `/documentation/ROADMAP.md` - Zukünftige Entwicklung
- `/documentation/CHANGELOG.md` - Vollständiges Änderungsprotokoll
